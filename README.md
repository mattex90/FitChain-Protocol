# FitChain Protocol

A decentralized fitness tracking and reward system that incentivizes consistent workout habits through blockchain-based tokens.

## Overview

FitChain Protocol gamifies fitness by rewarding users with tokens for completing workouts and maintaining consistency. Built on Stacks blockchain, it provides transparent, verifiable fitness achievements.

## Features

- **Workout Tracking**: Log workout sessions with intensity-based validation
- **Consistency Rewards**: Build streaks for multiplied token rewards
- **Token Locking**: Demonstrate long-term commitment for enhanced benefits
- **Achievement Verification**: All fitness milestones recorded on-chain

## Getting Started

1. Start a workout session with `start-workout`
2. Complete your workout with `finish-workout`
3. Build consistency streaks for bonus rewards
4. Withdraw or lock tokens based on your fitness goals

## Technical Specifications

- Base workout reward: 15 tokens
- Consistency multiplier: 3x per level (max 10 levels)
- Maximum supply: 5,000,000 tokens
- Dedication lock period: 576 blocks minimum
